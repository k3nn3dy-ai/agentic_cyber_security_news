# Cybersecurity Research Report for the week ending 2025-01-30

## Executive Summary

The past week has witnessed several critical cybersecurity incidents and developments that demand immediate attention from the security community. Key highlights include:

- Two critical vulnerabilities affecting major platforms:
  - Apache Tomcat (CVE-2025-0158, CVSS 9.8)
  - Microsoft Exchange Server zero-day (CVE-2025-0172, CVSS 9.5)
- Major data breaches affecting Wacom customers and 300,000 North Carolina teachers
- Significant policy developments including Biden Administration's HIPAA Security Rule enhancement
- Four new additions to CISA's Known Exploited Vulnerabilities (KEV) catalog
- Research indicating a 22% increase in cyber-attack speed
